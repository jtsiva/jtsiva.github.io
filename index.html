Recording Guidance for Screencasts

Created concrete recording guidance for instructors to help them create more accessible screencast videos. I created my own demonstration videos to clearly illustrate best practices including zooming in, changing cursor color and size, and using more descriptive language to describe on-screen actions.

Expanded on a set of guidance slides by creating a video that illustrated the different elements required for this type of learning experience. This served the two-fold purpose of establishing a common vocabulary as well as providing concrete exampels of what we were looking for.
